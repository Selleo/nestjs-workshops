## Description

An app for Selleo's workshops about NestJS

## Installation

NodeJS: 16.13.2

```bash
$ yarn
```

## Running the app

1. Docker-compose with 3rd party services

```bash
docker-compose up
```

2. NestJS

```bash
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
